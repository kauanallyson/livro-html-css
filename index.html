HTML5 e CSS3: Domine a web do futuro

Estudos práticos do livro • Projeto Mirror Fashion

Repositório com os códigos desenvolvidos durante a leitura do livro "HTML5 e CSS3: Domine a web do futuro" de Lucas Mazza. O objetivo é: dominar HTML5, CSS avançado (Flexbox, Grid, animações), boas práticas de acessibilidade e construção do projeto prático Mirror Fashion.

Tecnologias e Conceitos Estudados

HTML5 Semântica, formulários avançados, multimídia, acessibilidade
CSS3 Seletores avançados, Box Model, Flexbox, Grid, animações
Boas Práticas Código limpo, organização, responsividade, acessibilidade
